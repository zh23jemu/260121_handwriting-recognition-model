# 手写汉字数据集下载指南

## 一、推荐数据集

### 1. CASIA-HWDB（离线手写汉字数据库）
- **发布机构**：中国科学院自动化研究所（NLPR）
- **数据集特点**：包含大量手写汉字样本，适合训练CRNN模型
- **推荐版本**：HWDB2.0（包含行文本数据，适合CRNN模型训练）

### 2. 其他可选数据集
- **CASIA-OLHWDB**（在线手写汉字数据库）：包含在线手写轨迹数据
- **ICDAR手写汉字数据集**：国际文档分析与识别会议发布的数据集

## 二、下载地址

### 1. CASIA官方下载地址
- **主页面**：https://nlpr.ia.ac.cn/databases/handwriting/Home.html
- **下载页面**：https://nlpr.ia.ac.cn/databases/handwriting/Download.html

### 2. 下载步骤
1. 访问CASIA手写数据库下载页面
2. 注册/登录CASIA账号（需要学术机构邮箱）
3. 选择需要下载的数据集版本（推荐HWDB2.0）
4. 同意数据集使用协议
5. 开始下载数据集压缩包

## 三、数据集组织建议

### 1. 目录结构
```
handwriting-recognition/
└── dataset/
    ├── raw/              # 原始数据集存放目录
    │   └── HWDB2.0/      # HWDB2.0原始数据
    └── processed/        # 处理后数据集存放目录
        ├── train/        # 训练集
        ├── val/          # 验证集
        └── test/         # 测试集
```

### 2. 手动创建目录（PowerShell命令）
```powershell
mkdir dataset
mkdir dataset\raw
mkdir dataset\processed
mkdir dataset\processed\train
mkdir dataset\processed\val
mkdir dataset\processed\test
```

## 四、数据集预处理

### 1. 数据筛选
- 从HWDB数据集中筛选出1000个常用汉字样本
- 每个汉字保留适量样本（建议每个汉字至少100个样本）

### 2. 预处理步骤
- **灰度化**：将彩色图像转换为灰度图像
- **二值化**：使用自适应阈值将灰度图像转换为黑白图像
- **降噪**：使用中值滤波或高斯滤波去除噪声
- **归一化**：将图像大小归一化到统一尺寸（例如32×100像素）
- **数据增强**：旋转、缩放、平移等操作增加数据多样性

## 五、注意事项

1. 数据集下载可能需要学术机构邮箱注册
2. 数据集较大，建议使用稳定的网络连接下载
3. 下载后请验证文件完整性，避免解压失败
4. 请遵守数据集使用协议，仅用于学术研究
5. 处理后的数据集建议进行备份，避免数据丢失

## 六、替代方案

如果无法访问CASIA数据集，可以考虑以下替代方案：

1. **自行收集数据**：组织人员书写常用汉字，拍摄或扫描成图像
2. **使用公开的在线手写数据集**：例如Kaggle上的手写汉字数据集
3. **联系相关研究机构获取授权**：通过学术合作获取数据集访问权限
